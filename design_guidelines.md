# Kerala Economic Forum (KEF) Website Design Guidelines

## Design Approach
**Premium Startup-Style**: Combining Google 2025 design language with Apple website minimalism. Young, energetic, bold aesthetic targeting entrepreneurs, students, and institutions.

## Brand Colors
- **Primary Palette**:
  - Soft Red: #E46E6E
  - Yellow/Gold: #E7D26A  
  - Aqua/Teal: #6EC9C6
  - Dark Charcoal/Black: Typography
  - Clean White: Backgrounds

- **Visual Treatment**:
  - Soft gradients blending the 3-color palette
  - Subtle glassmorphism with blurred layers
  - Warm lighting with soft shadows
  - Neon glow accents on icons using brand colors

## Typography
- **Font Families**: Poppins, Inter, or Plus Jakarta Sans
- **Hierarchy**: Large bold typography for heroes, clean sans-serif throughout
- **Style**: Modern, readable, startup-centric

## Layout System
- **Spacing**: Generous whitespace, modern breathing room
- **Structure**: Premium card-based layouts, floating elements
- **Corners**: Rounded edges throughout
- **Shadows**: Soft, subtle elevation

## Core Components

### Hero Sections
- Large bold typography with animated overlapping circles (matching KEF logo concept)
- Smooth fade-ins and sliding transitions
- Parallax background images where applicable
- Floating elements with depth

### Cards & Modules
- Floating cards with soft shadows
- Hover lift effects with smooth transitions
- Glassmorphism treatment on selected components
- Modular, reusable design patterns

### Buttons
- Rounded corners, premium finish
- When on images: blurred backgrounds
- No additional hover states needed (component handles this)
- Clear CTAs with brand color accents

### Navigation
- Clean, minimal header
- Smooth transitions between pages
- Sticky behavior where appropriate

## Animation Strategy

### Required Animations (GSAP)
- **Scroll-triggered**: Fade-ins, slide transitions, scroll reveals
- **Parallax**: Hero images and layered elements
- **Hover Micro-interactions**: Card lifts, icon glows, zoom effects
- **Counters**: Animated metrics on homepage
- **Transitions**: Smooth gradient shifts between sections

### Specific Effects
- Acrylic/glassmorphism sections
- Overlapping circle animations (logo-inspired)
- Event card hover zoom
- Partner grid hover glow
- Collapsible sections with smooth expansion

## Page-Specific Design

### Home Page
- Hero with animated circles, large typography
- Interactive metrics counter
- Floating program cards
- Premium events slider
- Partner grid with hover effects

### About Us
- Side-by-side animated storytelling
- Leadership team cards with hover effects
- Gradient section backgrounds

### Programs & Startup Support
- Modular cards with scroll animations
- Icon animations with neon glow
- Vertical timeline layouts
- Collapsible sections

### Events & Membership
- Event cards with hover zoom
- Sort/filter UI
- Multi-tier pricing cards with highlighted popular plan

### Resources & Partners
- Blog-style article layout
- Download cards with icons
- Scrolling marquee for partners
- Glassmorphism annual report section

### Contact
- Minimal form design
- Map integration
- Bold typography, clear CTAs

## Images
**Hero Images**: Yes, use impactful imagery on key pages (Home, About, Events)
- Home hero: Dynamic entrepreneurship/community imagery with parallax effect
- About: Team/community photos showcasing Kerala's entrepreneurial ecosystem
- Events: Event photography showcasing KEF activities
- Campus Initiatives: Youthful, vibrant college imagery

**Supporting Images**: Program illustrations, startup visuals, workshop photos

## Visual Elements
- Light 3D floating shapes
- Animated startup-style icons
- Gradient overlays inspired by logo circles
- Modern iconography throughout
- Warm lighting effects

## Responsive Design
- Desktop-first with mobile optimization
- Touch-friendly interactions on mobile
- Adapted animations for mobile performance

## Brand Personality
Innovation • Entrepreneurship • Community • Youth Empowerment • Professionalism • Kerala's Economic Transformation
